    Приложение для учета личных доходов и расходов
Программа написана на Python3.11
 

Структура файлов:
 main.py        |Главный файл, точка входа в программу
 interface.py   |Файл, реализующий интерфейс взаимодействия с кошельком
 model.py       |Файл, реализующий взаимодействие с данными
 templates.py   |Шаблоны для длинных текстовых переменных
 config.py      |Параметры конфигурации
 data.txt       |Файл с данными. строки соотвествуют записям. Поля записей разделены символом •

Основные варианты сценариев навигации:
1. Вывод простого баланса:
    Вывод баланса -> Простой отчет
2. Вывод развернутого отчета:
    Вывод баланса -> Развернутый отчет
3. Добавление записи:
    Добавление записи -> (ввод полей записи. в случае верного их заполнения, их запись в файл)
4. Изменить поля записи:
    Редактирование записи -> Изменить поля записи -> 
        -> (ввести параметры поиска по полям в записях) -> (в случае нахождении ровно одной записи - ввод новых значений полей указанной записи)
5. Удаление записи:
    Редактирование записи -> Удалить запись -> 
        -> (ввести параметры поиска по полям в записях) -> (в случае нахождении ровно одной записи - удаление этой записи)
6. Поиск по записям:
    Поиск по записям -> (ввести параметры поиска по полям в записях) -> (получение списка записей, удовлетворяющих условиям поиска)
